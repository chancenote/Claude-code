# 찬스노트-코딩용어사전

코딩 초보자를 위한 프로그래밍 용어사전. 60개 내장 용어를 쉬운 비유와 예시 코드로 설명합니다.

## UI/UX 개선 요약

1. **디자인 시스템 전면 교체** — CSS 변수 기반 역할별(role-based) 색상 토큰 도입 (surface/text/border/accent/tag)
2. **Bento 카드 레이아웃** — 검색 결과를 정의/비유/코드/관련용어 섹션 카드로 분리, 2열 그리드 구성
3. **Command Palette 검색** — `Ctrl+K` / `/` 키 포커스, 최근 검색어 드롭다운, 입력 가이드 마이크로카피
4. **타이포그래피 위계 강화** — Pretendard 폰트, display/heading/subheading/body/caption/micro 6단계 계층
5. **다크모드 고대비 완성** — Slate 계열 팔레트, 코드블록/카드/태그 모두 고대비 처리
6. **접근성(A11y) 개선** — semantic HTML, aria-label/role, focus-visible 링, prefers-reduced-motion 존중
7. **마이크로 인터랙션** — 토스트 시스템(성공/에러/복사), 카드 hover 트랜지션, fade-in/slide-up/scale-in 모션
8. **검색기록 페이지 개선** — "최근 7일" 필터 추가, 상대시간 표시, 삭제 확인 모달, 빈 상태 추천 검색어 CTA
9. **브랜딩/SEO** — 타이틀 "찬스노트-코딩용어사전", OG/Twitter 메타, Pretendard 웹폰트
10. **푸터 명언** — Arthur C. Clarke 명언 영/한 정확 표기
