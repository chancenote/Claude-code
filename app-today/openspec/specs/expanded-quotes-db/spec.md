## ADDED Requirements

### Requirement: 100개+ 카테고리 명언 데이터셋 제공
시스템은 100개 이상의 카테고리에 걸쳐 3,000개 이상의 이중언어(한국어·영어) 명언을 `QUOTES_DB` 객체로 제공해야 한다(SHALL).  
각 명언은 `{ ko, en, author }` 형식이며, author는 `"한글이름 (English Name)"` 또는 `"○○ 속담 (○○ Proverb)"` 형식이어야 한다(SHALL).

#### Scenario: 기존 35개 카테고리 확장
- **WHEN** 검색어가 기존 카테고리(예: "희망", "성공", "인생")에 해당할 때
- **THEN** 해당 카테고리에 15개 이상의 명언이 존재해야 한다

#### Scenario: 신규 경영/비즈니스 카테고리 제공
- **WHEN** 검색어가 "경영", "기업가정신", "마케팅", "전략", "혁신", "스타트업" 중 하나일 때
- **THEN** 해당 카테고리에서 15개 이상의 명언이 반환되어야 한다

#### Scenario: 감성/치유 카테고리 제공
- **WHEN** 검색어가 "위로", "치유", "공감", "슬픔", "그리움" 중 하나일 때
- **THEN** 해당 카테고리에서 15개 이상의 명언이 반환되어야 한다

#### Scenario: 철학/인문 카테고리 제공
- **WHEN** 검색어가 "철학", "진리", "윤리", "역사", "문학" 중 하나일 때
- **THEN** 해당 카테고리에서 15개 이상의 명언이 반환되어야 한다

#### Scenario: 동기부여 카테고리 제공
- **WHEN** 검색어가 "자기계발", "습관", "목표달성", "마음챙김", "회복력" 중 하나일 때
- **THEN** 해당 카테고리에서 15개 이상의 명언이 반환되어야 한다

#### Scenario: 3개 명언 무작위 표시
- **WHEN** 어떤 카테고리를 검색하든
- **THEN** 해당 카테고리에서 무작위로 3개 명언이 선택되어 표시된다

### Requirement: 이중언어 필드 완결성
모든 명언은 `ko`(한국어)와 `en`(영어) 필드를 모두 가져야 한다(SHALL).  
한국어 원문 명언은 영어 번역을, 영어 원문 명언은 한국어 번역을 함께 제공해야 한다(SHALL).

#### Scenario: 한국어 명언 확인
- **WHEN** QUOTES_DB의 임의 항목을 확인할 때
- **THEN** `ko` 필드와 `en` 필드가 모두 비어있지 않아야 한다

#### Scenario: 저자 형식 확인
- **WHEN** QUOTES_DB의 임의 항목을 확인할 때
- **THEN** `author` 필드가 `"한글 (English)"` 또는 `"○○ 속담 (○○ Proverb)"` 형식이어야 한다

### Requirement: SYNONYM_MAP 신규 카테고리 커버
`SYNONYM_MAP`은 신규 카테고리와 관련된 동의어·유사어를 포함해야 한다(SHALL).  
예: "CEO" → ["경영", "리더십"], "명상" → ["마음챙김", "영혼"], "위로" → ["치유", "공감"]

#### Scenario: 비즈니스 관련 동의어 검색
- **WHEN** 검색어가 "CEO", "창업", "비즈니스", "사업", "회사", "경영인" 중 하나일 때
- **THEN** SYNONYM_MAP을 통해 관련 카테고리 명언이 반환되어야 한다

#### Scenario: 감성 관련 동의어 검색
- **WHEN** 검색어가 "눈물", "힘들다", "힐링", "위안" 중 하나일 때
- **THEN** SYNONYM_MAP을 통해 관련 카테고리 명언이 반환되어야 한다

### Requirement: KEYWORD_CATEGORIES 다양화
`KEYWORD_CATEGORIES` 배열은 신규 카테고리군을 반영해 최소 10개 이상의 테마 그룹을 제공해야 한다(SHALL).  
AI 추천 버튼 클릭 시 다양한 주제의 키워드가 노출되어야 한다.

#### Scenario: AI 추천 키워드 다양성
- **WHEN** AI 추천 버튼을 클릭할 때
- **THEN** 경영, 감성, 철학, 동기부여, 스포츠, 과학 등 신규 카테고리 키워드가 포함될 수 있어야 한다
